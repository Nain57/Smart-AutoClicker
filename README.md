## Introduction

Smart AutoClicker is an Android application to automate clicking for repetitive tasks. It allows you to capture an image on your screen to automatically click when this image is detected again. Unlike other auto clicker apps, this app isn't based on timers to execute auto clicking.

## Features

- Organize clicks by scenario
- Execute clicks or swipes
- Add an image condition from the screen
- Modify the tolerance for the condition detection
- Combine multiple conditions
- Configure the delay before the next click
- Modify the prority order of a click
