<p>Klick'r 是一款開源 Android 應用程式，旨在輕鬆自動化重複性工作。原名 Smart AutoClicker，Klick'r 結合獨特的影像辨識與傳統自動點擊功能，為各種自動化需求提供彈性方案。
</p>
<p>無論您是為遊戲內操作建立自動化、模擬使用者互動的測試人員，或需要重複點擊的一般使用者，Klick'r 都提供影像辨識以實作進階自動化，以及步驟簡單的「一般模式」滿足直覺的自動點擊需求。
</p>

<br>

## 主要功能:
* **點擊與滑動**：透過設定按壓時間、滑動時間與位置，精準自動化點擊與滑動。可在偵測到指定影像時觸發動作，與會變動的介面元素順暢互動。
* **進階自動化**：運用計數器操作、Android Intent、流程控制等進階功能強化腳本，獲得高度彈性。
* **觸發條件**：依影像辨識、計時器、計數器與 Android 廣播接收器等建立觸發條件，精準貼合您的自動化情境。
* **一般模式**：以簡潔設定體驗直覺的自動點擊，適合較單純、重複性高的工作。
* **教學**：透過互動式遊戲教學逐步學會 Klick'r，並運用其功能完成任務、破關遊戲。
* **開源**：作為開源專案，Klick'r 由社群持續改進。
